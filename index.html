Basic Cleaning Logic

TRIM

Menghapus "Spasi Hantu". Excel sangat sensitif terhadap spasi ekstra di awal, akhir, atau spasi ganda.

=TRIM(A2)
" Data " → "Data"
"A B" → "A B"

Standardisasi Huruf

PROPER Title Case
=PROPER("jakarta pusat")
UPPER All Caps
=UPPER("sku-01")
LOWER All Lower
=LOWER("Email@TEST.com")

Logika Potong (Extraction)

Potong Statis (Fixed Length)

LEFT Ambil Kiri
=LEFT(teks, jml_huruf)
  • teks: Sel yang mau dipotong.
  • jml_huruf: Berapa karakter?
=LEFT("SBY-01", 3)
// Hasil: "SBY"
RIGHT Ambil Kanan
=RIGHT(teks, jml_huruf)
  • teks: Sel yang mau dipotong.
  • jml_huruf: Berapa karakter?
=RIGHT("SBY-01", 2)
// Hasil: "01"
MID Ambil Tengah
=MID(teks, start, jml)
  • start: Mulai huruf keberapa?
  • jml: Ambil berapa huruf?
=MID("SBY-01", 5, 2)
// Hasil: "01"

Logika Dinamis (Dynamic Cut)

Kasus: Mengambil Nama Depan dari "Budi Santoso". Kita tidak bisa pakai angka mati (misal 4), karena nama orang beda-beda panjangnya.

Langkah 1: Pahami FIND
Kamus Syntax: FIND =FIND("cari_apa", di_sel_mana, [mulai_nomor])
1. cari_apa:
Teks yang dicari (misal spasi " ").
2. di_sel_mana:
Sel referensi (A2).
3. [mulai_nomor]:
Opsional. Start cari dari urutan keberapa.
=FIND(" ", "Budi Santoso")
→ 5
Langkah 2: Masukkan ke LEFT
=LEFT("Budi Santoso", 5 ) // Hasil: "Budi " (Ada spasi)

Solusi Akhir (Kurangi 1 agar spasi hilang):

=LEFT( A2, FIND(" ", A2) - 1 )

// Hasil: "Budi"

The Logic of Time

Konsep: Tanggal = Angka

Di Excel, tanggal hanyalah angka biasa yang diformat.
1 = 1 Jan 1900.

Tgl Akhir - Tgl Awal = Durasi Hari

Menggabungkan (DATE)

PENTING
Kamus Syntax =DATE(tahun, bulan, hari)

Hati-hati! Urutannya Tahun dulu, bukan Hari.

=DATE(2023, 12, 25)
// Hasil: 25/12/2023

Memecah Tanggal (Ekstraksi)

=DAY(sel) Output: 1-31

Mengambil Tanggal saja.

=MONTH(sel) Output: 1-12

Mengambil Bulan (Angka).

=YEAR(sel) Output: 4 Digit

Mengambil Tahun (YYYY).

Contoh Kasus: Expired (Tambah 1 Tahun)

=DATE( YEAR(A1)+1, MONTH(A1), DAY(A1) )

Teknik Substitusi (Ganti Teks)

SUBSTITUTE

Mengganti teks lama dengan teks baru. Sangat berguna untuk membersihkan karakter "sampah".

Kamus Syntax =SUBSTITUTE(teks, lama, baru, [ke_brp])
teks Sel awal (A2).
lama Teks yg mau dibuang (misal "-").
baru Teks pengganti (misal "").
ke_brp (Opsional) Urutan keberapa?
Studi Kasus 1: Standardisasi No HP
IN
0812-3456-7890
=SUBSTITUTE(A2, "-", "")
// Instance Num kosong = Hapus semua strip
OUT
081234567890

Fitur Rahasia: [ke_brp]

Bagaimana jika kita hanya ingin mengganti karakter tertentu saja? Misalnya mengganti tanda strip kedua menjadi tanda tambah, tapi biarkan strip pertama.

Teks Awal
"Kopi-Susu-Gula"
Rumus (Ganti strip ke-2)
=SUBSTITUTE(A1, "-", "+", 2)
Hasil: "Kopi-Susu+Gula"

Evolusi Penggabungan

1. Klasik (Manual)

JADUL

Ampersand (&)

A1 & " " & B1

CONCATENATE

=CONCATENATE(A1, " ", B1)

Ribet. Tidak bisa blok range (A1:A5), harus pilih satu-satu.

2. CONCAT

2019+
=CONCAT(A1:A5)
Kelemahan: Tidak punya fitur "Pemisah". Hasilnya akan menempel semua (BudiAndiSiti).

3. TEXTJOIN

WINNER
Kamus Syntax =TEXTJOIN(pemisah, abaikan_kosong, range)
1. pemisah Karakter pemisah (misal ", " atau spasi).
2. abaikan_kosong TRUE = Lompat sel kosong.
FALSE = Tulis sel kosong.
3. range Blok data (A1:A10).
=TEXTJOIN(", ", TRUE, A1:A10)
